# Minesweeper
A compact version of the popular game used on command line.
# Installation
Clone the repository with 
```git clone https://github.com/wilsonwei123/minesweeper```. 
Change to the directory
```cd minesweeper```
then run 
```g++ minesweeper.cpp -o minesweeper -std=c++20```
